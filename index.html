Overview

A portfolio built to stand out through both design and technical quality.

The site combines consistent branding, elegant spacing, subtle animation, reusable components, and functional JavaScript features in a way that feels more complete than a basic coursework website.

Premium Visual Style

A controlled colour palette, clean typography, glassmorphism details, gradients, and refined spacing create a stronger first impression.

Responsive Layout

Built with Grid and Flexbox to adapt smoothly across desktop, tablet, and mobile screen sizes without breaking the layout.

Interactive Experience

Includes theme switching, mobile navigation, reveal animations, project filtering, modal windows, FAQ behaviour, and form validation.
